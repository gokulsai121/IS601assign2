Assignment 2

Linux Commands

CD: Change Directory, mkdir: Make Directory, ls: List Files.

Vi Commands

i: Insert Mode, Esc: Get Out Of Insert Mode, :W: To Write, :Q: To Quite, :WQ: Write And Quite.

Git Commands

Git Init: Create Repository, Git Add Filename: Add The File To Get In, Git Commit -m "Message": Commit To Save Changes, Git Checkout -b Branchname: Make A New Branch And Checkout, Git Checkout Branchname: Checkout Branch, Git Merge Branchname: Merge The Branch
